Built an end-to-end AI-powered news aggregation and personalization platform that ingests updates from selected sources, generates concise digests, ranks stories per user, and prepares email-ready briefings.

Implemented authenticated user workflows, profile-based personalization, source management, pipeline orchestration, and production deployment with scheduled jobs and cloud database integration.

**Tech Stack:** Next.js, TypeScript, Clerk, OpenAI API, PostgreSQL/Neon, Resend, Docker, Python, Vercel.
